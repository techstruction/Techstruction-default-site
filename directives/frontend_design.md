# Directive: Frontend Design (Aesthetic Excellence)

## Goal
Create distinctive, production-grade frontend interfaces that avoid generic AI aesthetics.

## Design Thinking
1. **Tone Selection**: Explicitly pick an aesthetic (e.g., Brutalist, Luxury, Retro-futuristic).
2. **Typography**: Use distinctive Google Fonts pairs. Avoid Inter/Arial.
3. **Motion**: Focus on high-impact page load reveals and subtle hover states.
4. **Composition**: Break the grid. Use asymmetry and generous negative space.

## Orchestration Steps
1. Define the aesthetic vision and color palette.
2. Call `execution/frontend_factory.py` to scaffold the component directory.
3. Generate the HTML structure and Vanilla CSS/JS.
4. Refine details: add textures, meshes, and micro-animations.

## Execution Tools
- `execution/frontend_factory.py`

## Expected Output
- A cohesive, visually striking frontend component/page in a dedicated directory within `frontend-design/`.
